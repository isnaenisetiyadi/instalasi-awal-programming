Cara install composer di mac os

1.  Melalui terminal dan pastikan berada di direktori Downloads
2.  Download composer dengan perintah: curl -sS https://getcomposer.org/installer | php
3.  sudo mv composer.phar /usr/local/bin/
4.  sudo chmod 755 /usr/local/bin/composer.phar
5.  merubah nama composer.phar menjadi composer dengan perintah :
    sudo mv /usr/local/bin/composer.phar /usr/local/bin/composer

7.  pengujian dengan mengetik composer atau composer --version
